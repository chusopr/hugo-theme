# Chuso Theme for Hugo

This is a clean, elegant, but fully functional blog theme for Hugo. Here is a live [demo site](https://chuso.net) using this theme.  

It is based [cleanwhite theme](https://github.com/zhaohuabing/hugo-theme-cleanwhite) which is at the same time based on [huxblog Jekyll Theme](https://github.com/Huxpro/huxpro.github.io)
and [Clean Blog Jekyll Theme](https://github.com/BlackrockDigital/startbootstrap-clean-blog-jekyll).

This theme is basically a fork of _cleanwhite_ theme with some customizations made for my site [chuso.net](https://chuso.net).

Some of these customizations may be too much ad-hoc, so I cannot guarantee they will work with other sites that are not built exactly the same as mine. Similarly, I cannot guarantee backward or forward compatibility between versions.  
Basically, this is a personal project. You can use it, but at your own risk.

Some of the customizations include:

* Fixed typos.
* Sidebar in post pages too.
* I18n (multilingual support).
* Customize categories and tags to use my custom and messed-up implementation that makes taxonomies translatable.
* Fix invalid HTML.
* Add post's categories under the title.
* Use absolute URLs for cannonical links.
* Make markup semantics more meaningful.
* Update Font Awesome to make more icons available.
* Add client-side search.
* Use Staticman for comments with support for nested comments.
